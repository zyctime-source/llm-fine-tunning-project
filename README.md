# llm-fine-tunning-project

个人实验仓库：与 Cursor 协作，在数月内从零推进一个**以大模型微调为核心、最终落在安卓端**的落地项目，并全程留档。

## 背景与动机

我一直对大模型微调很感兴趣。近年小参数量的高质量模型越来越多（例如 Gemma 2B / 4B、Qwen3 0.8B / 2B 等），在端侧与手机上的部署空间也越来越现实。本仓库记录的是：**能否在有限周期内，把「微调 + 安卓落地」走通一条可发布的路径**。

## 项目目标（硬性约束）

- **必须包含**大模型微调环节（不是只做 RAG 或纯 API 套壳）。
- **最终形态**：模型或推理链路能落到**安卓手机**上；长期希望上架**国内安卓应用商店**，也不排除**微信小程序**等形态。

## 协作与时间线

计划在约 **2～3 个月**内，从想法到可演示、再到可发布的节奏推进；具体里程碑会随 shaping / PoC 结果调整。

## 过程记录

希望完整记录每一次与 Cursor 协作的过程与结论，覆盖从 **ideation → shaping → PoC → development → launch** 各阶段。详细日志与材料主要放在仓库的 `log/`、`shaping/` 等目录中，便于回溯与复盘。

## 现状说明

前方仍有大量**未知**：起步时产品形态与垂直场景未必清晰，会在迭代中与 Cursor 一起逐步收敛。本 README 会随项目进展更新。
